# Paper Materials

This directory contains paper drafts, figures, tables, and supplementary materials for the CultureDx study.

## Directory Structure

```
paper/
  drafts/          Paper draft versions (LaTeX/PDF)
  figures/         Publication-quality figures
  tables/          Result tables (LaTeX/CSV)
  supplementary/   Supplementary materials
  preprints/       Preprint versions (PDF)
```
